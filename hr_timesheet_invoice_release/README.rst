============================
HR Timesheet Invoice Release
============================

This is a glue module that makes hr_timesheet_invoice_release
work correctly in presence of the hr_timesheet_sheet module.

Installation
============

There is no specific installation instructions for this module.
It is autoinstalled when hr_timesheet_sheet and hr_timesheet_invoice_release
are both installed.
